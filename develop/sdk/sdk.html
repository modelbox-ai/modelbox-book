SDK API集成

SDK API集成

Modelbox提供了SDK API的方式，同时提供C++ SDK API接口和Python SDK API接口。

适用场景

通常场景基于ModelBox的AI应用作为独立进程运行，AI应用和周边系统应用交互通过http或者其他消息中间件进行通信。当AI应用需要作为组件被集成到其他系统进程中时，也可以直接使用ModelBox提供的SDK API进行集成开发，完成应用的集成。
